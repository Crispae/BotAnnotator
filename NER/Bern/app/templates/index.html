BERN2

An Advanced Neural Biomedical Named Entity Recognition and Normalization Tool (Sung et al., 2022)
DMIS Lab at Korea University

Features of BERN2

  • Recognition of 9 biomedical entity types along with normalized IDs
    (Gene, Disease, Chemical, Species, Mutation, Cell Line, Cell Type, DNA, and RNA)
  • Plain text or PMID as an input and annotated results in real time
  • RESTful API or local installation (documentation)
  • Pre-processed PubMed annotations (annotations)
